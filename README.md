Weather App

Простое приложение для просмотра погоды в любом городе мира.

О проекте
Приложение показывает текущую температуру, погодные условия и иконку для выбранного города. Данные получает с OpenWeatherMap API.

Функционал
- Поиск города
- Температура в °C
- Описание погоды
- Иконка погоды

Технологии
- HTML5
- CSS3 (адаптив)
- JavaScript (ES6+)
- OpenWeatherMap API


Файлы
`index.html` - разметка
`style.css` - стили
`script.js` - логика
